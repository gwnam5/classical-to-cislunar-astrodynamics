# Classical-to-Cislunar Astrodynamics Toolkit

## Goal

This project builds a reproducible astrodynamics toolkit that connects classical two-body orbital mechanics to cislunar multibody dynamics.

The goal is not to propose a completely new theory, but to develop a strong working understanding of the core concepts, algorithms, and dynamical language needed for graduate-level astrodynamics research.

## Learning Spine

1. Two-body problem
2. Keplerian propagation
3. Classical orbital elements and state vectors
4. Coordinate frames
5. Maneuvers and targeting
6. Relative motion and linearization
7. Numerical propagation and perturbations
8. Circular restricted three-body problem
9. State transition matrix and differential correction
10. Cauchy-Green tensor and principal stretching directions
